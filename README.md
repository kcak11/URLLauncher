# <img src="https://cdn.kcak11.com/URLLauncher/icons/icon128.png?v=001" width="32" height="32" valign="middle"> URL Launcher

A sleek and efficient Google Chrome extension designed to provide quick access to your most-used URLs directly from a popup window.

## 🚀 Features

- **Quick Launch**: Access your favorite links instantly from the browser toolbar.
- **Instant Search**: Quickly filter through your links using the built-in search bar with keyboard navigation support (Up/Down arrows to highlight, Enter to open).
- **Dark & Light Themes**: Toggle between themes to suit your preference, with automatic system synchronization.
- **Customizable Size**: Adjust the popup dimensions via the options page to fit your screen and link list.
- **High-Quality UI**: Modern design with glassmorphism effects, smooth transitions, and a premium "URL" logo.

## 🛠 Installation

1. Clone or download this repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** using the toggle in the top right corner.
4. Click on **Load unpacked** and select the `URLLauncher` directory.
5. Pin the URL Launcher extension to your toolbar for easy access.

## ⚙️ Configuration

Your links are managed in the `links.json` file located in the root directory.

### Customizing Dimensions

To change the popup size:
1. Right-click the extension icon and select **Options**.
2. Enter your desired width and height.
3. Click **Save Settings**.

## 📖 Keyboard Shortcuts

When the popup is open and the search input is focused:
- **Arrow Down**: Highlight the next link in the list.
- **Arrow Up**: Highlight the previous link.
- **Enter**: Launch the currently highlighted link (or the first result if none is highlighted).
